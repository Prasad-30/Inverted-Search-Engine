# 🔍 Inverted Search Engine (C)

A C-based project that builds an inverted index using hash tables and linked lists to efficiently search words across multiple text files, showing file locations and frequency.

## 🚀 Features

* Create database from multiple files
* Search words across files
* Display indexed data
* Update with new files
* Save & restore database

## 🛠️ Tech Used

C, Hash Tables, Linked Lists, File Handling

## ⚙️ Run

```bash
make
./a.out file1.txt file2.txt
```
